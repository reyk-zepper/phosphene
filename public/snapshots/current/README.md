# Phosphene Published Redacted Snapshot

These files are synthetic and redacted. They are not live telemetry.

They are intended for Phosphene Node Observer snapshot loading. Phosphene loads `manifest.json`, the listed `*.synthetic.json` trace bundles, and `validation-report.json` from the served `/snapshots/current/` path.

Seed path in the repo: `public/snapshots/current/`

Served path after deploy: `/snapshots/current/`
