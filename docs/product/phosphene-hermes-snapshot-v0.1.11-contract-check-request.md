Rules:

- Do not use `event_id`; use `trace_id`.
- Do not use `type`; use `event_type`.
- Every event's `run_id` must equal the file's `metadata.id`.
- Every non-root `parent_event_id` must reference another event's `trace_id` in the same file.
- There must be exactly one root event where `parent_event_id` is `null`.
- `links` must be omitted or be an array of objects with `label` and `href`.
- Do not emit link strings like `["trace://..."]`.
- Link `href` values must use `trace://...` only.
- Optional string fields must be omitted when no value applies.

Forbidden keys anywhere in trace bundles:

